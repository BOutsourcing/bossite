4Human - Evolution -
Employee Monitoring Tool

Long-term cooperation on enterprise project during which we were responsible for software development, project management, analysis, maintenance and b2b support. Our team has worked on various technical aspects of this project and supported the company's growth on Norwegian market (over 1000 clients)

Human Resources

Evolution is developed using Agile methodology and Kanban method. Project is managed using Atlassian's Jira. It employs technologies such as: Java, PHP, MySQL, git, SAML. It offers integration with external systems using RESTful API as well as with Norwegian e-Government system.

Application is built so scale horizontally with failover in mind. Code base contains more than 3M lines of code. Every new feature goes through code review, automated and manual testing process with Continuous Integration.

Project details

Solutions

This unique HR solution covers employee on-boarding, shift planning and replacements, clock-in/out, absence registration and follow-up, injury registration, performance management, work agreements, Altinn integration, contracts and handbooks.

Technologies

Apache, PHP, MySql, Java, RESTful, Kubernetes, Jenkins, Cucumber, RestAssured, Geb, Selenium Webdriver, Gatling, Terraform, Consul, Chef Fiddler, Mitmproxy, Maven,IntelliJ, Jackson, Guava, Gulp, Grunt, RAML, Swagger, Nexus

Tools

JIRA, GitLab, Docker, Slack

Team

8 PHP developers, 2 Application Tester - QA, 1 Infrastructure Engineer, 1 Business Analyst, 2 Support Representatives

About the Client

Evolution is one of the most used HRMS in Norway which provides a unique web-based management tool for a comprehensive employee monitoring. The solution ensures that all the employees in the organization are taken care of and that all employees information are available at any time or place.
