Human Resources

Long-term cooperation on enterprise project during which we have been responsible for software development, project management, analysis, maintenance and b2b support. Our team has worked on various technical aspects of this project and supported the company's growth in the Norwegian market (over 1000 clients).

Evolution is developed using Agile methodology and Kanban method. The project is managed using Atlassian's Jira. It employs technologies such as PHP, MySQL, git, SAML. It offers an integration with external systems using RESTful API as well as an integration with the Norwegian e-Government system.

Application is built to scale horizontally with failover in mind. Codebase contains more than 3M lines of code. Every new feature goes through code review, automated and manual testing process with Continuous Integration.

Project details

Solution

This unique HR solution covers employee on-boarding, shift planning and replacements, clock-in/clock-out, absence registration and follow-up, injury registration, performance management, work agreements, company restructuring, Altinn integration, contracts and handbooks. All these data are available in the web and mobile version along with 90+ reports.

Technologies

Apache, PHP, MySql, Java, RESTful, Kubernetes, Jenkins, Cucumber, RestAssured, Geb, Selenium Webdriver, Gatling, Terraform, Consul, Chef Fiddler, Mitmproxy, Maven, IntelliJ, Jackson, Guava, Gulp, Grunt, RAML, Swagger, Nexus

Tools

JIRA, GitLab, Docker, Slack

Team

Over the years the team grew according to our client’s needs in order to support the expanding customer base.

8 PHP developers, 2 Application Tester - QA, 1 Infrastructure Engineer, 1 Business Analyst, 2 Support Representatives

About the client

Evolution is one of the most used HRMS in Norway which provides a unique web-based management tool for comprehensive employee monitoring. The solution ensures that all the employees in the organization are taken care of and that all employees' information is available at any time or place. Evolution is a modern and user-friendly management tool with many convenient features for efficient and better follow-up of staff.
