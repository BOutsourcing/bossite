Project details

Solutions

MySiteShop is an intuitive and easy to use website builder. This project consists of a template catalog and an editing tool - Mosaic. This solution was created as a simple and efficient tool consisting of functionalities such as text, news and link editor, file manager, gallery, calendar, catalog, rss feed, social media etc.

Technologies

PHP, HTML/Bootstrap

Tools

Jira, Confluence, GitHub

Team

One of the key accomplishments that we are proud of is that each team member is capable of working on every part of the system.
We achieved this through the use of documentation, standards in UI and in development, agile processes and lots of hard work.

4 back end developers, 1 front end developer, 1 QA specialist, 1 Business Analyist/Scrum Master, 1 customer support agent

About the customer

MySiteShop is a website market with premade website templates. MySiteShop is an easy-to-use website builder, with clean and responsive design, suitable for any business. Regardless of their skill level, users can build a fully-functional website. Project is based on a simplified editor and variety of features where users can easily add content and publish their website in just a few steps.
