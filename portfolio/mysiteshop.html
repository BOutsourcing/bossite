For this website builder project, we created a dedicated team that mapped and analyzed the requirements, designed the solution, planned and implemented the entire software and infrastructure.

Our team provided continuous delivery and developed new features and integration with the external member management system. Throughout the entire project life-cycle, we provided technical and customer support.

For the legacy website builder system - WebComputing, our team provided maintenance, technical support and migration services.

Project details

Solution

MySiteShop is an intuitive and easy-to-use website builder. This project consists of a template catalog and an editing tool - Mosaic. The solution was created as a simple and efficient tool consisting of functionalities such as text, news and link editor, file manager, gallery, calendar, catalog, RSS feed, social media etc.

Technologies

PHP, HTML/Bootstrap

Tools

Jira, Confluence, GitHub

Team

4 back end developers, 1 front end developer, 1 QA specialist, 1 Business Analyst/Scrum Master, 1 customer support agent

About the customer

MySiteShop is a website market with premade website templates and an easy-to-use website builder, with clean and responsive design, suitable for any business. Regardless of their skill level, users can build a fully-functional website. The project is based on a simplified editor and a variety of features where users can easily add content and publish their website in just a few steps.
