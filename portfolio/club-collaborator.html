Project details

Solutions

This unique member management system for Rotary clubs and districts provides an easy to use set of features such as member directory, extensive reports, meeting registration functionalities, archives, club and district management and a role based access structure.

Technologies

PHP, MySql, HTML/Bootstrap

Tools

Jira, Confluence, GitHub

Team

1 full stack developer, 1 Account Manager, 1 Business Analyst/Scrum Master

About the customer

Club Collaborator is a membership management software for Rotary clubs with an official Rotary International license. The project is based on a set of organizational tools and features necessary for any Rotary Club or District to manage their member directory, meeting attendance and reporting. Club Collaborator is integrated with Rotary International and convenient for simple membership overview and faster reporting. Clean and well-organized access rights structure based on different roles and high-level security make this software suite unique and very efficient.
